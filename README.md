# 🚀 Zero to Kubestronaut

A hands-on journey through the complete Cloud Native ecosystem —
from Linux fundamentals to full platform engineering.

This repository documents every step of the 
[Golden Kubestronaut](https://www.cncf.io/training/kubestronaut/) path:
real implementations, POCs, and working configurations
across 15 certifications and 7 phases.
